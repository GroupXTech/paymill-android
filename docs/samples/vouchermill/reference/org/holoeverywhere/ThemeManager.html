Class Overview

ThemeManager for using different themes in activity, dialogs, etc.
It uses the principle of binary flags. For example, you can get a dark theme on the fullscreen mixing the two flags:
DARK | FULLSCREEN

Default themes map for different flags:

DARK LIGHT MIXED
no other flags Holo_Theme Holo_Theme_Light Holo_Theme_Light_DarkActionBar
FULLSCREEN Holo_Theme_Fullscreen Holo_Theme_Light_Fullscreen Holo_Theme_Light_DarkActionBar_Fullscreen
NO_ACTION_BAR Holo_Theme_NoActionBar Holo_Theme_Light_NoActionBar Holo_Theme_Light_DarkActionBar_NoActionBar
NO_ACTION_BAR | FULLSCREEN Holo_Theme_NoActionBar_Fullscreen Holo_Theme_Light_NoActionBar_Fullscreen Holo_Theme_Light_DarkActionBar_NoActionBar_Fullscreen
WALLPAPER Holo_Theme_Wallpaper Holo_Theme_Light_Wallpaper Holo_Theme_Light_DarkActionBar_Wallpaper
WALLPAPER | FULLSCREEN Holo_Theme_Fullscreen_Wallpaper Holo_Theme_Light_Fullscreen_Wallpaper Holo_Theme_Light_DarkActionBar_Fullscreen_Wallpaper
WALLPAPER | NO_ACTION_BAR Holo_Theme_NoActionBar_Wallpaper Holo_Theme_Light_NoActionBar_Wallpaper Holo_Theme_Light_DarkActionBar_NoActionBar_Wallpaper
WALLPAPER | NO_ACTION_BAR | FULLSCREEN Holo_Theme_NoActionBar_Fullscreen_Wallpaper Holo_Theme_Light_NoActionBar_Fullscreen_Wallpaper Holo_Theme_Light_DarkActionBar_NoActionBar_Fullscreen_Wallpaper
DIALOG Holo_Theme_Dialog Holo_Theme_Dialog_Light Holo_Theme_Dialog_Light
DIALOG_WHEN_LARGE Holo_Theme_DialogWhenLarge Holo_Theme_DialogWhenLarge_Light Holo_Theme_DialogWhenLarge_Light_DarkActionBar
DIALOG_WHEN_LARGE | NO_ACTION_BAR Holo_Theme_DialogWhenLarge_NoActionBar Holo_Theme_DialogWhenLarge_Light_NoActionBar Holo_Theme_DialogWhenLarge_Light_DarkActionBar_NoActionBar

You may remap themes for certain flags with method map(int, int) where first arg - flags, second - theme resource
